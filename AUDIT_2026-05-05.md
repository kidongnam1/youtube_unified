## 권장 패치 순서

영향도와 사용자 빈도 기반 우선순위:

1. **LME.py `run_ui_test` (P1)** — 빠른 fix (1줄 변경 가능). 즉시 사용자 체감 개선.
2. **SQM `main_webview.py` startup polling (P2)** — 기동 시 매번 발생, 모든 사용자에게 영향. `wait_for_api`/`kill_zombie_on_port`를 백그라운드 스레드로 이동.
3. **SQM `engine_modules/database.py` DB 락 재시도 (P1 조건부)** — 락 빈도 낮으나 발생 시 영향 큼. 핸들러 레이어에서 thread 위임 강제 또는 DB 호출 자체를 worker queue로 이동.
4. **SQM `ollama_manager.start_ollama_server` (P2)** — local LLM 사용 사용자 한정. 호출 시점에 thread 래핑.

6. **`menu_mixin._show_manual` 등 P3 그룹** — `subprocess.run` → `subprocess.Popen` 일괄 치환 (Linux/Mac 환경 한정 영향).
